## 1. Visión General del Sistema

El CBF Planner opera sobre un principio central de **diseño inverso (backward design)**:

```
┌─────────────────────────────────────────────────────────────────┐
│                  PRINCIPIO DE DISEÑO INVERSO                    │
│                                                                 │
│   1. ¿Qué quiero que el estudiante LOGRE al final?              │
│              ↓                                                  │
│   2. ¿Cómo EVIDENCIARÉ que lo logró? (evaluación)              │
│              ↓                                                  │
│   3. ¿Qué EXPERIENCIAS y ACTIVIDADES lo llevarán allí?         │
│                                                                 │
│   "Nosotros diseñamos. El docente enseña."                     │
└─────────────────────────────────────────────────────────────────┘
```

Este enfoque implica que el docente **no parte de los contenidos** sino de los **resultados esperados de aprendizaje**, y desde ahí construye hacia atrás las experiencias que llevan al estudiante a alcanzarlos.

### 1.1 Filosofía Integradora

Toda la planificación pedagógica en CBF está atravesada por tres ejes transversales:

```
         ┌──────────────────┐
         │   COSMOVISIÓN    │
         │    BÍBLICA       │
         └────────┬─────────┘
                  │
     ┌────────────┼────────────┐
     ▼            ▼            ▼
┌─────────┐ ┌─────────┐ ┌─────────┐
│ACADÉMICO│ │ VALORES │ │  AÑO   │
│         │ │   Y     │ │TEMÁTICO │
│Contenido│ │ACTITUDES│ │  2026:  │
│Curricular│ │         │ │ Pureza  │
└─────────┘ └─────────┘ └─────────┘
```

Cada Logro, Indicador, Experiencia Significativa y Evaluación debe poder conectarse con al menos uno de estos tres ejes.

## 2. Jerarquía Curricular

El sistema curricular del CBF se organiza en niveles anidados, de lo más amplio a lo más específico:

```
NIVEL 0: AÑO LECTIVO
│
├── NIVEL 1: ÁREA (ej. Lenguaje, Matemáticas, Cosmovisión Bíblica)
│   │
│   ├── NIVEL 2: GRADO (ej. 8°, 9°)
│   │   │
│   │   ├── NIVEL 3: TRIMESTRE (Trimestre 1, 2, 3)
│   │   │   │
│   │   │   ├── NIVEL 4: LOGRO DEL TRIMESTRE
│   │   │   │   │
│   │   │   │   ├── NIVEL 5: TEMÁTICA 1
│   │   │   │   │   ├── Indicador de Logro 1
│   │   │   │   │   └── Actividades Evaluativas
│   │   │   │   │
│   │   │   │   ├── NIVEL 5: TEMÁTICA 2
│   │   │   │   │   ├── Indicador de Logro 2
│   │   │   │   │   └── Actividades Evaluativas
│   │   │   │   │
│   │   │   │   └── (Temáticas adicionales...)
│   │   │   │
│   │   │   └── NIVEL 4: EXPERIENCIA SIGNIFICATIVA
│   │   │       (Proyecto integrador del trimestre)
│   │   │
│   │   ├── NIVEL 3: TRIMESTRE 2
│   │   └── NIVEL 3: TRIMESTRE 3
```


### 2.2 Dos Modelos de News: Estándar vs. Lengua

El sistema CBF reconoce **dos variantes estructurales** del News según el área:

```
┌──────────────────────────────────────────────────────────────────────┐
│  MODELO A — NEWS ESTÁNDAR                                             │
│  (Español, Matemáticas, Ciencias, Cosmovisión, etc.)                │
├──────────────────────────────────────────────────────────────────────┤
│  TRIMESTRE                                                           │
│  ├── Temática 1                                                      │
│  │   ├── Indicador de Logro                                         │
│  │   └── Actividades Evaluativas                                     │
│  ├── Temática 2                                                      │
│  │   ├── Indicador de Logro                                         │
│  │   └── Actividades Evaluativas                                     │
│  └── EXPERIENCIA SIGNIFICATIVA (sección al final)                   │
│      ├── Descripción general del proyecto                            │
│      ├── Pasos distribuidos en el tiempo                             │
│      └── Rúbrica vinculada                                           │
│                                                                      │
│  La ES es UNA sola e integradora de todas las temáticas.            │
└──────────────────────────────────────────────────────────────────────┘

┌──────────────────────────────────────────────────────────────────────┐
│  MODELO B — NEWS DE LENGUA                                            │
│  (Language Arts / Inglés, y otras lenguas extranjeras)              │
├──────────────────────────────────────────────────────────────────────┤
│  TRIMESTRE                                                           │
│  ├── Competencias declaradas                                         │
│  │   (Sociolingüística, Lingüística, Pragmática)                    │
│  ├── Habilidades (Speaking, Listening, Reading, Writing)            │
│  ├── Operadores Intelectuales                                        │
│  │   (Deducir, Generalizar, Sintetizar, Retener, Evaluar...)        │
│  │                                                                   │
│  ├── INDICADOR 1 — Speaking                                          │
│  │   ├── Principio Bíblico con versículo PROPIO del indicador       │
│  │   ├── Enunciado del Indicador (inglés + traducción al español)   │
│  │   ├── Experiencia de Aprendizaje Significativa EMBEBIDA          │
│  │   │   (dentro del indicador, con descripción, grupo y criterios) │
│  │   └── Actividades estándar recurrentes:                          │
│  │       Dictados / Vocabulary / Quiz / Cambridge One /             │
│  │       Plan lector / PET Preparation                              │
│  │                                                                   │
│  ├── INDICADOR 2 — Listening  (misma estructura)                    │
│  ├── INDICADOR 3 — Reading    (misma estructura)                    │
│  └── INDICADOR 4 — Writing    (misma estructura)                    │
│                                                                      │
│  Cada indicador tiene su propia ES embebida.                        │
│  La rúbrica aparece al final, organizada por habilidad.             │
└──────────────────────────────────────────────────────────────────────┘
```

**Comparación de los dos modelos:**

| Dimensión | Modelo A — Estándar | Modelo B — Lengua |
|-----------|--------------------|--------------------|
| Organización | Por temáticas de contenido | Por habilidades lingüísticas |
| Número de ES | Una sola al final | Una por indicador/habilidad |
| Principio bíblico | Versículo del año (compartido) | Versículo propio por indicador |
| Competencias declaradas | Implícitas en el logro | Explícitas al inicio del News |
| Operadores intelectuales | Opcionales | Siempre presentes |
| Actividades recurrentes | Varían por docente | Set estándar fijo (Dictados, Quiz, Cambridge One, etc.) |
| Idioma del indicador | Español | Inglés + traducción al español |
| Tamaño del grupo en ES | Variable | Declarado explícitamente (ej. "2 estudiantes") |

## 3. Logros e Indicadores de Logro

### 3.1 ¿Qué es un Logro?

El **Logro** es el enunciado que describe el **resultado de aprendizaje esperado al finalizar el trimestre** en un área específica. Es la meta grande, el horizonte al que apunta toda la planificación del período.

**Características de un Logro bien formulado:**

```
┌─────────────────────────────────────────────────────────────────┐
│  ANATOMÍA DE UN LOGRO                                           │
│                                                                 │
│  [VERBO COGNITIVO] + [CONTENIDO/COMPETENCIA] +                 │
│  [CONTEXTO/CONDICIÓN] + [DIMENSIÓN VALORATIVA opcional]        │
│                                                                 │
│  Ejemplo:                                                       │
│  "Construir textos narrativos coherentes aplicando             │
│   normas ortográficas básicas, demostrando respeto             │
│   por las ideas propias y las de sus compañeros."             │
│    ───────┬───────   ─────────┬────────────────   ──────┬───── │
│           │                  │                          │       │
│       VERBO            CONTENIDO              DIMENSIÓN │       │
│      COGNITIVO         + CONDICIÓN            VALORATIVA│       │
└─────────────────────────────────────────────────────────────────┘
```

**Verbos cognitivos recomendados por nivel (Taxonomía de Bloom adaptada CBF):**

| Nivel | Verbos | Uso |
|-------|--------|-----|
| Recordar | Identificar, reconocer, listar, nombrar | Temáticas introductorias |
| Comprender | Explicar, describir, relacionar, comparar | Temáticas de desarrollo |
| Aplicar | Construir, usar, implementar, demostrar | Actividades prácticas |
| Analizar | Distinguir, examinar, contrastar, inferir | Proyectos complejos |
| Evaluar | Valorar, juzgar, defender, criticar | ES y reflexiones finales |
| Crear | Diseñar, producir, elaborar, componer | Experiencias Significativas |

### 3.2 ¿Qué es un Indicador de Logro?

El **Indicador de Logro** es la evidencia observable y medible de que el estudiante está avanzando hacia el Logro del trimestre. **Cada Temática tiene su propio Indicador de Logro.**

```
LOGRO DEL TRIMESTRE (meta grande)
         │
         ├── TEMÁTICA 1 → INDICADOR 1 (evidencia parcial)
         ├── TEMÁTICA 2 → INDICADOR 2 (evidencia parcial)
         └── TEMÁTICA N → INDICADOR N (evidencia parcial)
                                │
                    La suma de Indicadores
                    evidencia el Logro completo
```

**Diferencias clave entre Logro e Indicador:**

| Dimensión | LOGRO | INDICADOR DE LOGRO |
|-----------|-------|-------------------|
| Alcance | Todo el trimestre | Una temática específica |
| Nivel | Macro / holístico | Micro / observable |
| Redacción | Amplio, integrador | Preciso, verificable |
| Cantidad | 1 por área/trimestre | 1 por temática |
| Evaluación | Experiencia Significativa | Actividades de la temática |

**Ejemplo de relación Logro → Indicadores:**

```
ÁREA: Lenguaje | GRADO: 1° | TRIMESTRE 1

LOGRO:
"Construir textos cortos y narraciones sencillas aplicando el uso
correcto de las mayúsculas y el punto, demostrando comprensión de
los personajes y lugares, y participando con respeto por las ideas
de sus compañeros en las actividades grupales."
         │
         ├── TEMÁTICA 1: El abecedario, mayúsculas, punto y coma
         │   INDICADOR: "Escribe oraciones simples usando
         │              mayúsculas al inicio y punto final,
         │              identificando sus partes correctamente."
         │
         └── TEMÁTICA 2: Texto instructivo: receta, tarjeta
             INDICADOR: "Relaciona la descripción de eventos y
                        personas con la secuencia de pasos,
                        usando frases cortas y el abecedario
                        para comunicar ideas claramente."
```

### 3.3 Formulación de Indicadores: Guía Práctica

Un Indicador de Logro debe responder a estas preguntas:

```
1. ¿QUÉ hace el estudiante?     → Verbo observable en acción
2. ¿CON QUÉ lo hace?            → Contenido o recurso específico
3. ¿PARA QUÉ / en qué contexto? → Propósito o situación
4. ¿CON QUÉ CALIDAD?            → Condición de desempeño (opcional)
```

**Lista de verificación de un buen Indicador:**

- [ ] Usa un verbo que describe una acción observable
- [ ] Es específico para la temática (no genérico)
- [ ] Puede medirse o evidenciarse en una actividad
- [ ] Es alcanzable en el tiempo de la temática
- [ ] Se conecta claramente con el Logro del trimestre
- [ ] Integra, cuando es posible, una dimensión actitudinal o de cosmovisión

### 3.4 El Principio Bíblico por Indicador (Modelo B — Lengua)

En el Modelo B (Language Arts / Inglés), cada Indicador de Logro va precedido de un **Principio Bíblico** con versículo propio. Este elemento es adicional al versículo del año y es específico para la habilidad lingüística del indicador:

```
ESTRUCTURA DEL PRINCIPIO BÍBLICO POR INDICADOR

┌────────────────────────────────────────────────────────────────┐
│  INDICADOR N — [Habilidad: Speaking / Listening / Reading /    │
│                            Writing]                            │
├────────────────────────────────────────────────────────────────┤
│  PRINCIPIO BÍBLICO:                                            │
│  [Título del principio — conectado temáticamente]              │
│  [Referencia bíblica] (versión en inglés)                     │
│  "[Cita textual del versículo]"                               │
├────────────────────────────────────────────────────────────────┤
│  INDICADOR:                                                    │
│  [Enunciado en inglés]                                         │
│  [Traducción al español]                                       │
├────────────────────────────────────────────────────────────────┤
│  EXPERIENCIA SIGNIFICATIVA EMBEBIDA                            │
│  [Descripción + tamaño de grupo + criterios específicos]       │
└────────────────────────────────────────────────────────────────┘
```

**Ejemplo real — Indicador 1 (Speaking), 9th Grade, T1 2025:**

```
PRINCIPIO BÍBLICO:
"God's plan: A dream worth waiting for!"
Génesis 50:20 (NIV) — "You intended to harm me, but God intended
it for good to accomplish what is now being done..."

INDICADOR — Speaking:
"Present information, findings, and supporting evidence clearly,
concisely, and logically such that listeners can follow the line
of reasoning and the development about personal achievements,
introducing one another and describe workers qualities, using
simple and continuous tenses and dynamic and stative verbs."

ES EMBEBIDA — "How do my dreams line up with God's plan?"
Vision Board: hablar de sueños comparándolos con un líder bíblico.
Incluir tiempos verbales del indicador. Grupo: 2 estudiantes.
```

**Propósito del Principio Bíblico por Indicador:**
El versículo no es decorativo. Debe actuar como **ancla temática** que le da sentido espiritual y contextual a la habilidad lingüística que se trabaja. La ES embebida debe conectar explícitamente con ese versículo.

### 3.5 Competencias y Operadores Intelectuales (Modelo B)

El Modelo B declara explícitamente al inicio del News:

```
COMPETENCIAS (marco de Language Arts / Cambridge):
• Sociolingüística  — Uso del lenguaje en contexto social
• Lingüística       — Dominio estructural del idioma
• Pragmática        — Uso del lenguaje con intención comunicativa

OPERADORES INTELECTUALES (procesos cognitivos del período):
• Deducir    — Extraer conclusiones de información dada
• Generalizar — Aplicar conceptos a nuevas situaciones
• Sintetizar — Integrar información en un todo coherente
• Retener    — Fijar vocabulario y estructuras en la memoria
• Evaluar    — Juzgar la calidad de textos o producciones
```

Estos operadores deben aparecer en los enunciados de los Indicadores y en el diseño de las ES embebidas.

## 4. Experiencias Significativas y Proyectos

### 4.1 ¿Qué es una Experiencia Significativa?

La **Experiencia Significativa (ES)** es la actividad culminante del trimestre. Es el momento donde el estudiante **integra y demuestra** todo lo aprendido durante el período a través de una tarea auténtica, creativa y significativa.

```
┌─────────────────────────────────────────────────────────────────┐
│              CARACTERÍSTICAS DE UNA ES                          │
│                                                                 │
│  ✦ INTEGRADORA: conecta todos los Indicadores del trimestre    │
│  ✦ AUTÉNTICA: simula o es un reto del mundo real               │
│  ✦ SIGNIFICATIVA: tiene propósito más allá del aula            │
│  ✦ PROCESUAL: se construye a lo largo del trimestre            │
│  ✦ EVALUABLE: tiene rúbrica con criterios claros               │
│  ✦ BÍBLICA: puede conectarse con el eje de Cosmovisión         │
└─────────────────────────────────────────────────────────────────┘
```

### 4.2 Modalidades de Experiencia Significativa

No existe una modalidad preferida. Todas las siguientes son válidas y equivalentes en el sistema:

```
                    EXPERIENCIAS SIGNIFICATIVAS
                           │
          ┌────────────────┼────────────────┐
          │                │                │
    PRODUCTOS          PROCESOS        ACTUACIONES
          │                │                │
   ┌──────┴──────┐  ┌──────┴──────┐  ┌──────┴──────┐
   │ Libro       │  │ Investigación│  │ Debate      │
   │ ilustrado   │  │ de campo    │  │ académico   │
   │             │  │             │  │             │
   │ Periódico   │  │ Experimento │  │ Exposición  │
   │ mural       │  │             │  │ oral        │
   │             │  │ Proyecto de │  │             │
   │ Revista     │  │ impacto     │  │ Obra de     │
   │ digital     │  │ comunitario │  │ teatro      │
   │             │  │             │  │             │
   │ Maqueta     │  │ Portafolio  │  │ Podcast o   │
   │ / modelo    │  │ reflexivo   │  │ video       │
   └─────────────┘  └─────────────┘  └─────────────┘
         │                │                │
    Evaluada con     Evaluada con     Evaluada con
    rúbrica de       rúbrica de       rúbrica de
    producto         proceso          actuación
```

### 4.3 Estructura de una Experiencia Significativa en el Sistema

Toda ES en el CBF Planner se documenta con los siguientes componentes:

```
┌─────────────────────────────────────────────────────────────────┐
│  ESTRUCTURA DE UNA EXPERIENCIA SIGNIFICATIVA                    │
├─────────────────────────────────────────────────────────────────┤
│  1. NOMBRE                                                      │
│     Título evocador que describe la ES                         │
│                                                                 │
│  2. DESCRIPCIÓN GENERAL                                         │
│     Párrafo que explica qué harán los estudiantes,             │
│     qué evaluará y cómo se conecta con el Logro               │
│                                                                 │
│  3. LOGRO ASOCIADO                                              │
│     El Logro del trimestre que la ES evalúa                    │
│                                                                 │
│  4. INDICADORES QUE INTEGRA                                     │
│     Lista de los Indicadores de Logro que evidencia            │
│                                                                 │
│  5. PASOS / ACTIVIDADES SECUENCIALES                            │
│     Las etapas del proceso, distribuidas en el tiempo          │
│     (una por semana o por bloque de semanas)                   │
│                                                                 │
│  6. RÚBRICA DE EVALUACIÓN                                       │
│     Criterios, niveles y puntajes específicos                  │
│                                                                 │
│  7. FECHA DE PRESENTACIÓN / ENTREGA                             │
│     Momento de culminación y evaluación final                  │
└─────────────────────────────────────────────────────────────────┘
```

### 4.4 Los Pasos de una ES: Diseño Procesual

La ES **no es una actividad de un solo día**. Se construye a lo largo del trimestre mediante pasos secuenciales:

```
SEMANA 1    SEMANA 2    SEMANA 3    SEMANA 4    SEMANA 5    SEMANA N
   │            │           │           │           │           │
   ▼            ▼           ▼           ▼           ▼           ▼
[PASO 1]    [PASO 2]    [PASO 3]    [PASO 4]    [PASO 5]   [ENTREGA]
Activación  Exploración  Producción  Revisión    Afinado    Presentación
  inicial   del tema     inicial    y feedback  del trabajo  y evaluación
     │            │           │           │           │           │
     └────────────┴───────────┴───────────┴───────────┴───────────┘
                         PROCESO CONTINUO
                     (evaluado en cada paso)
```

**Principio clave:** El docente evalúa **proceso Y producto**, no solo el resultado final.

### 4.5 Diferencia entre News Project y Experiencia Significativa

En el CBF Planner, el término **"News Project"** se refiere al **documento trimestral** (el News) que contiene la Experiencia Significativa. No son lo mismo:

```
NEWS (documento)
│
├── Encabezado (docente, trimestre, versículo del año)
├── Tabla de Temáticas e Indicadores
├── EXPERIENCIA SIGNIFICATIVA ← aquí vive la ES
│   ├── Descripción del proyecto
│   ├── Pasos secuenciales
│   └── Rúbrica vinculada
└── Vocabulario y recursos de apoyo
```

## 5. Rúbricas

### 5.1 ¿Qué es una Rúbrica?

Una **rúbrica** es un instrumento de evaluación que describe explícitamente los niveles de desempeño para cada criterio que se evalúa. Permite que tanto docentes como estudiantes comprendan **qué se espera** y **cómo se mide la calidad**.

```
┌─────────────────────────────────────────────────────────────────┐
│                    ANATOMÍA DE UNA RÚBRICA                      │
├──────────────────┬──────────────┬──────────────┬───────────────┤
│    CRITERIO      │  NIVEL 1     │  NIVEL 2     │   NIVEL 3     │
│  (¿Qué evalúo?) │ (Básico/Inic)│ (En proceso) │ (Avanzado)    │
├──────────────────┼──────────────┼──────────────┼───────────────┤
│ Criterio 1       │ Descripción  │ Descripción  │ Descripción   │
│ Criterio 2       │ observable   │ observable   │ observable    │
│ Criterio 3       │ de lo que    │ de lo que    │ de lo que     │
│ ...              │ hace/muestra │ hace/muestra │ hace/muestra  │
│ Criterio 8       │ el estudiante│ el estudiante│ el estudiante │
└──────────────────┴──────────────┴──────────────┴───────────────┘
```

### 5.2 Estructura de la Rúbrica en el Sistema CBF

## 8. Escala de Valoración

### 8.1 Descripción de Niveles de Desempeño

El sistema CBF define claramente qué significa cada nivel para orientar tanto a docentes como a estudiantes:

```
NIVEL 5 — SUPERIOR (5.0 / 90-100)
─────────────────────────────────
El estudiante supera los criterios establecidos.
Demuestra dominio profundo del contenido, aplica el
conocimiento en situaciones nuevas y complejas, y puede
explicar o enseñar lo aprendido. Muestra iniciativa,
creatividad y excelencia en todos los aspectos evaluados.

NIVEL 4 — ALTO (4.0-4.9 / 70-89)
──────────────────────────────────
El estudiante cumple todos los criterios de manera sólida.
Demuestra comprensión clara del contenido, comete errores
menores que no afectan el desempeño general. Su trabajo
evidencia esfuerzo consistente y calidad sostenida.

NIVEL 3 — BÁSICO (3.0-3.9 / 60-69)
────────────────────────────────────
El estudiante cumple los criterios mínimos esperados.
Demuestra comprensión básica del contenido con algunas
imprecisiones. Su desempeño es aceptable pero requiere
refuerzo en aspectos específicos.

NIVEL 2 — BAJO (2.0-2.9 / 40-59)
──────────────────────────────────
El estudiante no alcanza todos los criterios mínimos.
Muestra comprensión parcial del contenido con errores
frecuentes. Requiere apoyo y acompañamiento significativo
del docente para avanzar.

NIVEL 1 — MUY BAJO (1.0-1.9 / 0-39)
─────────────────────────────────────
El estudiante no demuestra comprensión del contenido.
No alcanza los criterios establecidos. Requiere
intervención pedagógica urgente y plan de mejoramiento.
```

## 10. Glosario de Términos del Sistema

|---------|-------------------------------|

## Apéndice A: Checklist para un News Completo

```
ANTES DE PUBLICAR UN NEWS, VERIFICA:

LOGRO
□ El Logro está redactado con verbo de acción cognitiva
□ El Logro es alcanzable en el trimestre
□ Conecta con el eje bíblico/actitudinal

TEMÁTICAS E INDICADORES
□ Cada temática tiene su propio Indicador de Logro
□ El Indicador usa verbo observable
□ El Indicador es específico (no genérico)
□ Las actividades evaluativas están vinculadas a las temáticas

EXPERIENCIA SIGNIFICATIVA
□ La ES está claramente descrita (qué, cómo, para qué)
□ Los pasos están distribuidos en el tiempo
□ La ES integra los Indicadores del trimestre
□ La rúbrica está vinculada o incluida

RÚBRICA
□ Tiene 8 criterios definidos
□ Cada criterio tiene 5 niveles descritos
□ Los niveles tienen descriptores observables (no solo adjetivos)
□ La escala corresponde a la sede (1.0-5.0 o 0-100)

PUBLICACIÓN
□ Versículo del año incluido
□ Fechas del trimestre correctas
□ Vocabulario vinculado
□ Documento revisado por coordinación
```

## Apéndice B: Errores Comunes y Cómo Evitarlos

```
ERROR 1: Confundir Logro con Indicador
─────────────────────────────────────
✗ MAL: "Logro: Identifica las partes de la oración."
       (demasiado específico, es un indicador)
✓ BIEN: "Logro: Construye textos coherentes aplicando
        normas gramaticales básicas con creatividad."

ERROR 2: Indicador no observable
────────────────────────────────
✗ MAL: "Comprende la importancia del punto."
       (¿cómo se observa "comprende"?)
✓ BIEN: "Usa el punto correctamente al redactar
        oraciones de al menos 5 palabras."

ERROR 3: ES diseñada al final del trimestre
────────────────────────────────────────────
✗ MAL: Planificar la ES cuando ya se enseñaron las temáticas.
✓ BIEN: Diseñar la ES PRIMERO (backward design), luego
        las temáticas que preparan para ella.

ERROR 4: Rúbrica con descriptores vagos
────────────────────────────────────────
✗ MAL: Nivel 3: "El trabajo es bueno."
✓ BIEN: Nivel 3: "El texto incluye inicio, nudo y desenlace.
        Usa mayúsculas al inicio de oración en el 70% de los casos."

ERROR 5: Evaluar solo el producto final
────────────────────────────────────────
✗ MAL: Solo calificar la presentación final de la ES.
✓ BIEN: Asignar peso a los pasos del proceso (20% del total)
        y al producto final (30% del total).
```
