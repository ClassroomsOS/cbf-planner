# CBF Planner

Sistema de Planeación Docente — ClassroomOS

## Stack
- React 18 + Vite 5
- Supabase (Auth + PostgreSQL)
- GitHub Pages

## Desarrollo local

```bash
npm install
npm run dev
```

## Deploy
Automático al hacer push a `main` via GitHub Actions → GitHub Pages.

## Módulos actuales
- ✅ Auth (email + Google)
- ✅ Perfil docente (multi-colegio, school_id)
- ✅ Generador de guías CBF
- ✅ Historial de guías
- 🚧 Calendario escolar con festivos
- 🚧 Editor completo de actividades por día
